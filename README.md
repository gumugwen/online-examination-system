# 在线考试系统

#### 介绍
本网站是利用IDEA和MYSQL数据库开发的在线考试系统。主要针对教师出卷、出题、选择参考班级、发布考试以及被选择的参考班级的学生参加考试并生成考试记录的需求进行设计，最终实现教师能够自主发布考试，为考卷出题，并要求指定班级在规定时间内参加考试，相关的学生能够在考试时间内参加考试，并且考试结束后能够查看考试分数的功能。同时管理人员可以管理年级信息、专业信息、班级信息、学生信息以及教师等。
